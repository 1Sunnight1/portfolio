
## Инструкция по использованию 

### 1. Интерактивный ввод

```
Проверка MX-записей доменов email-адресов
Введите email-адреса (по одному на строку, пустая строка для завершения):
--------------------------------------------------
Email: test@gmail.com     ← введите первый email
Email: user@yandex.ru     ← введите второй email  
Email:                    ← нажмите Enter на пустой строке
```

### 2. Результаты

```
Результаты проверки:
--------------------------------------------------
test@gmail.com: домен валиден
user@yandex.ru: домен валиден
```


### 3. Статусы проверки

| Статус | Описание |
| :-- | :-- |
| `домен валиден` | MX-записи найдены ✅ |
| `домен отсутствует` | Домен не существует ❌ |
| `MX-записи отсутствуют` | Нет почтовых серверов ❌ |
| `некорректный формат` | Нет символа `@` ❌ |
| `ошибка проверки` | Проблемы с DNS ❌ |

### 4. Логика работы

```
1. email.split('@')[1] → извлекает домен
2. dns.resolver.resolve(domain, 'MX') → DNS-запрос
3. Анализ ответа → статус
```
