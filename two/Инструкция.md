

## Инструкция по использованию скрипта

### 1. Настройка скрипта

```
TOKEN = ""  # ← Вставьте токен
CHAT_ID = ""  # ← Вставьте chat_id
```

### 2. Подготовка файла

Создайте(пропишите сообщение в) файл `message.txt` (в той же папке, где скрипт)


### 3. Результаты

**✅ Успех**:

```
{'ok': True, 'result': {'message_id': , ...}}
```

Сообщение появится в чате мгновенно.

**❌ Ошибка 401 Unauthorized**: Неверный TOKEN — получите новый.

**❌ Ошибка "chat not found"**: Неправильный CHAT_ID или бот не в чате.

**❌ "message.txt не найден"**: Создайте файл в той же папке.
